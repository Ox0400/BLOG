Centos 安装python 2.7.x
yum install python-2.7.0 
先别着急复制粘贴！
首先，安装必要的库，否则等你安装之后你会发现特别蛋疼，import zlib 报错，https也报错，yum module报错，sudo: python: command not found等等等等。

如需sudo权限，请随时sudo 运行，或者sudo su 直接使用root用户。

yum install zlib
yum install zlib-devel
yum install bzip2-devel
yum install openssl-devel
yum install ncurses-devel
#yum install sqlite-devel
yum install openssl-devel

wgethttps://www.python.org/ftp/python/2.7.9/Python-2.7.9.tar.xz
chmod +x Python-2.7.9.tar.xz
tar xvf Python-2.7.9.tar.xz
cd Python-2.7.9
./configure --prefix=/usr/local/python2.7 #设置安装路径，防止和系统自带2.6.6冲突。
make && makealtinstall #应该等同于 make && make install吧？

/usr/local/python2.7/bin/python # 测试一下，发现安装成功啦

which  python  #找到系统自带python2.6.6可执行文件的路径
ls /usr/bin/python* #这一步可以忽略不计
mv /usr/bin/python /usr/bin/python.bak #备份原有python(2.6.6)，防止后面步骤出错

ls -al /usr/local/python2.7/bin/python #可应该有 python python2python2.7 python2.7-config 等等，可以忽略
rm /usr/bin/python #删除系统自带的python，上面其实已经mv为.bak文件了，可以忽略。
ln -s /usr/local/python2.7/bin/python /usr/bin/python#创建软链接、硬链接我也不知道，反正就是创建类似windows的快捷方式吧


# 如果没有easy_install，从下面安装python包管理工具 easy_install
sudo wget --no-check-certificatehttps://bootstrap.pypa.io/ez_setup.py -O - | python
unzip setuptools-15.2.zip
cd setuptools-15.2
sudo python setup.py install
cd ..
sudo rm -rf setuptools-15.2

#安装完python后，easy_install已经存在。替换系统已有的"可执行文件路径"
find /usr/local/python2.7 | grepeasy_install #找到刚安装的easy_install
which easy_install #找到原有的easy_install路径
rm /usr/bin/easy_install #删除原有的easy_install，也可以mv为.bak文件(2.6.6)
/usr/local/python2.7/bin/easy_install requests#用刚安装的easy_install测试一下
ln -s/usr/local/python2.7/bin/easy_install-2.7 /usr/bin/easy_install#创建软链

# 测试一下easy_install
sudo easy_install requests
sudo /usr/local/python2.7/bin/easy_install-2.7simplejson
sudo easy_install xmltodict

安装完毕。
