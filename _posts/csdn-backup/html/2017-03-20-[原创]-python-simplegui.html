大家可能在Linux上或者Windows上用python在做几个小游戏，但是网上很多帖子很多教程都是导入一个simplegui模块，翻遍了百度都是吹水骗访问量的，根本不是。
学编程，要学会用google，呵呵，你懂得。。。
首先，下载simplegui.zip这个包，是一个zip（windows格式的），没关系，下载好。
如果你是windows：
  1. 解压，里面有一个setup.py 
  2. win+R ，输入cmd，回车
  3. 控制台输入python setup.py install
  4. 这样就安装成功了，如果提示安装失败或者你打开python IDE： import simplegui发现moudle no defind ，没关系，直接把simplegui.py复制粘贴到C:\Pyton27\Lib\site-pages\ (依个人电脑，一般都是这个位置)就可以，可能路径拼写错误，因为我装的是linux-ubuntu。或者粘贴到Lib也可以。然后试着import一下。
如果你是Linux（我的是ubuntu12.04）
  1. 解压 -- linux-ubuntu 解压zip文件
  2. 解压后，接着输入：cd simplegui-0.1.0 
  3. sudo python setup.py install
不过在安装之前，先安装python-ck ：
sudo apt-get install python-tk
如果还是安装失败，直接复制到python路径就可以。
          4. sudo cp simplegui.py /usr/lib/python2.7
....
到这里就好了，不过咋结束之前，试着import一下噢。
