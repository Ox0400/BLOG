用SSH远程执行python命令，一直抱错：UnicodeEncodeError:'ascii' codec can't encode characters in position
解决办法，在项目第一个python中重新设置默认编码

import sys
reload(sys)
sys.setdefaultencoding( "utf-8" )


无论python写什么程序，一定sys.setdefaultencoding，一定sys.setdefaultencoding，一定sys.setdefaultencoding
