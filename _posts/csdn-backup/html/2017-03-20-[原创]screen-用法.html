之前发过一篇《Linux 终端/Screen查看历史输出 》，没写怎么用。最近发现公司很多人都是在xshell中打开10+个连接,甚至有20多个......

screen
screen -S name  --- 创建一个终端管理器
screen -ls          --- 查看现有的终端管理器
screen -r name   --- 重新打开终端管理器（可以理解为最大化
screen -d name  ---最小化已经打开的终端管理器（用命令行最小化
screen -dr name ---最小化打开的终端管理器并重新打开（可以理解为最大化
screen -dms namecommand ---临时创建一个终端管理器，并执行命令

进去screen后操作快捷键
ctrl + A + C 创建一个终端
ctrl + A + D 最小化终端管理器
ctrl + A + N 切换到下一个终端
ctrl + A +P 切换到上一个终端
ctrl + A + A 切换到上一次使用的终端（类似windows上的ALT+TAB
ctrl + D 关闭终端/ 退出终端管理器 

还可以从外部创建 screen, 然后发送指令

cd /usr/local/elasticsearch
screen -dm es
screen -ls
screen -S es -X stuff './bin/elasticsearch\n'
