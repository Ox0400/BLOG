import requests as req
res = req.get("https://www.google.com.hk" ,proxies={"https":"http://127.0.0.1:8087", "http":"http://127.0.0.1:8087"},  verify=True)
1、访问https协议可能会遇到SSLError，设置verify 为True/False即可
2、代理时，切记，要设置代理网址的协议类型！！飘红位置，因为代理端口是http协议，而访问地址是https协议，特别注意此处。
否则如下错误(未知协议)：SSLError: [Errno 1] _ssl.c:503: error:140770FC:SSL routines:SSL23_GET_SERVER_HELLO:unknown protocol
