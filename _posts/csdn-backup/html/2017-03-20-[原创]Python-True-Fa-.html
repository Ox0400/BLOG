在python中，任何对象都可以判断其真假值：True，False
在if或while条件判断中，下面的情况值为False：
1.None
2.Flase
3.数值为0的情况，如：0,0.0,0j
4.所有空序列，如：'',(),[]
5.所有空mapping，如：{}
6.instances of user-defined classes, if the class defines a __bool__() or __len__() method,
   when that method returns the integer zero or bool value False.

All other values are considered true — so objects of many types are always true.(所有的其他值都可以认为True --- 所以，许多类型的对象总为True)

很多时候我们会使用find这个函数，比如：
if find 'OK' in 'task:\t101\t' continue; else you code ...
很多时候我们的思维会认为find找不到值为-1，负数!=False 。负数=True，负数 !== True
