set在python中是一个常用的内置函数，作用是清除重复项，但是返回的类型是set类型
>>> ls = ['a','b','ac','ab','a','c','d']
>>> ls_1=set(ls)
>>> type(ls_1)
>>> ls_1
set(['a', 'ac', 'b', 'd', 'c', 'ab'])
>>> a=ls_1
>>> a
set(['a', 'ac', 'b', 'd', 'c', 'ab'])
>>> b=set('abzbs')
>>> a
set(['a', 'ac', 'b', 'd', 'c', 'ab'])
>>> b
set(['a', 's', 'b', 'z'])
>>> a|b #a或者b都没有重复的元素
set(['a', 'ac', 'b', 'd', 's', 'z', 'ab', 'c'])
>>> a&b #a或者b都存在的元素
set(['a', 'b'])
>>> a^b #a和b都没有重复的元素
set(['ac', 'ab', 'd', 'c', 's', 'z'])
>>> a-b #去掉a中和b相同的元素
set(['c', 'ac', 'ab', 'd'])
>>> a+b
