1、下载mongodb-src-r2.6.8.zip，解压到 D:\Program Files\mongodb-src-r2.6.8
2、下载python，2.7.x版本最合适
3、下载scons-2.3.4-setup.exe，双击安装。在cmd中可以直接输入scons，出现如下：
scons: *** No SConstruct file found.
File "c:\python27\lib\site-packages\scons-2.3.4-py2.7.egg\scons-2.3.4\SCons\Script\Main.py", line 920, in _main
说明安装成功
4、cd mongodb-src-r2.6.8 ##注意，文件夹下如下
5、scons all 
时间比较长，耐心等待~预计需要2.5个小时，你可以出去吃个饭，遛遛街，睡个午觉。

6、mongod 看看效果

7、我们试着连接一下mongodb，
再开一个终端，输入mongo

服务器显示如下：
8、Ctrl + C 关掉服务端，客户端会报连接错误。可见服务端和客户端都有一个定时检测机制。服务端检测有多少客户端链接，客户端检测跟服务端连接状态。

安装MongoDB python支持，easy_install pymongo
