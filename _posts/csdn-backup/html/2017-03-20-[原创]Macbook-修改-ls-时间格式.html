Macbook  下修改时间显示格式.
# Linux 下修改 .bashrc 或者 .profile ,添加 exportTIME_STYLE='+%Y-%m-%d%H:%M:%S' 即可

Macbook 可以使用 ls -lT.

格式依旧不太直观, 安装 `coreutils`, 使用 `gls`
1. brew install coreutils
2. 修改 .bashrc 或者 .profile 
添加: alias ls="gls -G"
添加: alias ll="ls -al"
3. source ~/.bashrc
