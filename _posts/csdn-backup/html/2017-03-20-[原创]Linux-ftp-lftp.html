用了一下，linux上的ftp真感觉是渣。。。(可能是我肤浅)
lftp挺好的:
sudo apt-get install lftp #(size:450kb)
fltp 000.000.000.000 -u username (你的ftp文件地址，用户名，回车输入密码即可。)

用法: lftp [OPTS]
`lftp' 是在 rc 文件执行后 lftp 执行的第一个命令
 -f           执行文件中的命令后退出
 -c            执行命令后退出
 --help              显示帮助信息后退出
 --version           显示 lftp 版本后退出
 其他的选项同 `open' 命令
 -e            在选择后执行命令
 -u [,]  使用指定的用户名/口令进行验证
 -p           连接指定的端口
               主机名, URL 或书签的名字
