昨天玩游戏，觉得太卡，于是弄了一个扩虚拟内存的软件。
但是呢，我没有内存卡，手机剩余容量也就200多MB吧。
打开这个软件，默认200缓存，我点了确定，软件退出了。
我以为失败了，又重新进去，写成100，又退出了。
接着手机挂了，怎么挂的呢，我也不知道。
反正重启了，而且再也进不去了，一直停留在这个页面

大不幸啊，想着把卡装会旧手机诺基亚1050吧，猜想到剪卡了！
果不其然，即使套在卡槽上也识别不出来。
废了，擦！
今天想到我原来高过安桌在Linux上的ADK，命令(adb)这个什么的。
于是，想着怎么让手机进入开发模式，【电源】 +【音量+】【电源】 +【音量-】 都不行，所幸乱按一通，擦，我也不知道怎么进入的...
里面报了一个错误，faild mount /efs.
搞了半天，哎，仇人，进入 在这个模式下选择恢复系统默认即可。
我清除用户数据回复出厂设置，然后清楚缓存，重新开机即可。(建议手机里面装上百度一键Root，然后在windows上删除预装软件。。。。）

---reboot system now        重启手机（每次刷机完毕都要选择此项以重启手机）

---apply SDcard：update.zip        安装存储卡中的update.zip升级包（你可以把刷机包命名为update.zip然后用这个选项直接升级）

---wipe data / factory reset        清除用户数据并恢复出厂设置（刷机前必须执行的选项之一）

---wipe cache partition        清除系统缓存（刷机前必执行之一，另外系统出了问题也可尝试此选项，一般能够解决）

---install zip from SDcard        从SDcard上安装zip刷机包（可以执行任意名称的zip升级包，不限制升级包名称，刷机也从此去）

---backup and restore       备份和还原系统

---mounts and storage       挂载和存储选项

---advanced       高级设置
